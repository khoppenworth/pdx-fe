Welcome to Electronic Regulatory Information System (eRIS) of EFDA


iLicense

iLicense is an online application that allows importers, exporters, wholesalers and manufacturers to apply for the certificate of competency and FMHACA staff to manage these applications online.

iRegister

iRegister is an online application which allows importers to apply for and receive medicine registration certificate to import medicines online and EFDA staff to manage these applications online.

iImport

iImport is an online application which allows importers to apply for and receive permits to import medicines online and EFDA staff to manage these applications online.
